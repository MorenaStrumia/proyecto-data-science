# proyecto-data-science
Proyecto final - Data Science
Proyecto Data Science

Este repositorio contiene el desarrollo del proyecto final de Data Science.

Se implementó un modelo de regresión utilizando Random Forest para predecir la facturación a partir de variables comerciales.

El análisis incluye:

Limpieza de datos
Selección de variables
Entrenamiento del modelo
Evaluación con métricas
Interpretación de resultados
